# GoFit – Fitness & Nutrition Web Application

## Project Description

**GoFit** is a web application designed to help users improve their health and fitness through **personalized workout and nutrition plans**.

The platform allows users to create an account, enter their **physical information** (height, weight, activity level) and receive **customized training and meal recommendations**.

Users can track their progress, follow structured workout programs and access **video-based exercise guidance**. The application will also include **subscription-based access to premium workout programs and exercise video libraries**.

Administrators will be able to manage users, workout programs, nutrition plans and uploaded training resources.

## Main Features

### 1. User Authentication

Users can create accounts and log in securely.

**Features:**

- User registration  
- Secure login using **Spring Security**  
- **Password encryption**  
- **Role-based access** (client / admin)

### 2. User Profile

Each user has a personal profile containing their **fitness data**.

**Information stored in the profile:**

- Height  
- Weight  
- Age  
- Activity level  
- Fitness goal *(lose weight / build muscle / maintain)*

Users can:

- Update their profile  
- View recommended plans  

### 3. Personalized Workout Plans

Based on the **user profile**, the system will suggest workout plans.

Examples include:

- Beginner full body  
- Weight loss workout  
- Muscle building program  

Workout plans include:

- Exercise list  
- Number of sets and repetitions  
- Optional training videos  

### 4. Nutrition Plan Recommendations

Users receive **nutrition recommendations** based on their goal.

Examples include:

- Calorie target  
- Example daily meals  
- Macro distribution suggestions  

Administrators can **create and edit nutrition plans**.

### 5. Workout Video Library

The platform includes a collection of **exercise videos uploaded by administrators**.

Users can:

- Browse exercises  
- Watch workout demonstrations  
- Follow structured training sessions  

Administrators can:

- Upload videos  
- Edit workout descriptions  
- Organize workouts by category  

### 6. Subscription System (Premium Content)

Some workout programs and videos are available only through **subscriptions**.

Example subscriptions:

- **Basic** – free workouts  
- **Premium** – full workout library  
- **Program packages** *(e.g., 15-session glutes program)*  

Users can:

- View subscriptions  
- Activate a subscription  
- Access premium workouts  

### 7. Progress Tracking

Users can track their **fitness progress**.

Examples include:

- Weight history  
- Completed workouts  
- Training streak  

### 8. Admin Panel

Administrators manage the platform.

Admins can:

- Manage users  
- Add or edit workout plans  
- Upload workout videos  
- Create nutrition plans  
- Manage subscriptions
